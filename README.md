# Trabajo Práctico Integrador – Eje I: Procesamiento Digital de Señales

## Descripción

El presente trabajo tiene como objetivo aplicar conceptos de procesamiento digital de señales para la generación, preprocesamiento y análisis de un dataset sintético de audio.
---

## Metodología

El trabajo se desarrolla en cuatro etapas:

1. **Generación de dataset**
2. **Preprocesamiento**
3. **Análisis espectral (FFT)**
4. **Extracción de features**
